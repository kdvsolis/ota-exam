# Email Classifier

## Description
This project is a simple email classifier that reads emails from a CSV file and classifies whether they are newsletters.

## Setup
1. Clone the repository: `git clone https://github.com/yourusername/email_classifier.git`
2. Navigate into the directory: `cd email_classifier`
3. Install the dependencies: `pip install -r requirements.txt`
4. Run the server: `python manage.py runserver`

## Usage
1. Open `http://localhost:8000/classify/` in your browser.
2. Upload your CSV file containing the emails.
3. Click the 'Upload and Classify' button to classify the emails.
4. You will see the results of the classification on the next page.

## Testing
Run the tests: `python manage.py test`
